# godeconz

... is a binding for the deconz Rest and Websocket API
It has exchangeable HTTP Client (default Resty) and Logger (default standard go logger)

## Features
- [ ] Alarm Systems 
- [ ] Configuration 
- [ ] Groups 
- [x] Lights 
- [ ] Rules 
- [ ] Scenes 
- [ ] Schedules 
- [x] Sensors
- [ ] Button Events
- [ ] Touchlink
- [ ] Websocket
